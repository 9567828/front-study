# Web Storage API

  • 자바스크립트로 웹브라우저에 데이터를 보관할 수 있는 공간
  • 데이터를 key/value 형태로 저장해 놓을 수 있다
  • 로컬 스토리지는 웹브라우저를 종료하더라도 데이터가 유지된다
  • 세션 스토리지는 웹브라우저를 종료하면 데이터가 삭제된다 (웹브라우져가 켜져있는 동안만 유지되는 데이터)
  • 데이터는 문자열로 저장 된다

# 가위바위보 게임 만들어보기

  • 웹페이지 상에서 동작하는 간단한 가위바위보 게임을 하나 만든다
  • 해당 웹페이지에 최초로 접속했을 때는 사용자의 이름을 입력 받아야 한다
  • 가위바위보를 진행할 때 마다 소지금과 전적이 계속 변한다
  • 웹 브라우져를 껏다 켜도 소지금과 전적이 유지되도록 만들어라

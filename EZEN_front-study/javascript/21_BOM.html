# BOM(Browser Object Model)

  • window 객체
  • 웹 브라우저에 직접 접근하여 기능을 활용 할 수 있는 객체
  • DOM도 BOM의 내부에 존재하는 객체이다
  • window는 생략이 가능하다(너무 당연해서)

# BOM 내부의 여러 객체들

  • window: 웹브라우저 최상위 객체
  • window.document: 현재 페이지의 HTML구조를 담고있는 객체
  • window.history : 웹브라우저의 페이지 이동 내역을 담고있는 개체
  • window.screen: 웹브라우저가 표시되는 화면에 대한 정보
  • window.location: 웹브라우저의 현재 위치에 대한 정보
  • window.navigator: 웹브라우저 자체에 대한 정보

# window.setTimeout(callbackFn, millisec)

  • 설정한 시간 만큼 기다린 후에 콜백 메서드를 한 번 실행한다
  • 시간은 밀리초 단위로 전달해야 한다
  • 설정된 타임아웃을 삭제하기 위해서는 설정당시 생성된 인스턴스를 clearTimeout()에 전달해야 한다

# window.setInterval(callbackFn, millisec)

  • 설정된 시간 마다 콜백 함수를 실행한다
  • 설정된 인터벌을 삭제하기 위해서는 설정 당시 생성된 인스턴스를 clearInterval()에 전달해야 한다

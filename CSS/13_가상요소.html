# 가상 요소(Pseudo Element)

  • 해당 요소의 특정 부분만 선택하는 선택자
  • 선택자 뒤에 ::을 붙여 원하는 부분을 선택한다
  • ::first-letter - 첫 번째 글자만 선택하기
  • ::first-line - 첫 번째 줄만 선택
  • ::selection - 블록 지정한 부분만 선택
  • ::before - 해당 요소의 바로 전
  • ::after - 해당 요소의 바로 다음

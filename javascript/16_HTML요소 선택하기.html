# Document Object Model (DOM)

  • 웹 브라우저의 HTML코드를 해석하여 생성해둔 객체
  • DOM을 통해 HTML의 모든 요소에 접근할 수 있다

# 원하는 요소를 쉽게 선택하기 위한 document의 메서드들

  • document.getElementByID(id) : id로 원하는 요소를 선택한다
  • document.getElementTagName(tagName) : 해당 태그명의 모든 요소를 선택한다
  • document.getElementClassName(className) : 해당 클래스 속성을 지닌 모든 요소를 선택한다

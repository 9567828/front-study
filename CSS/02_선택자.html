# CSS 선택자

  • tagName: 해당 태그 이름의 모든 요소를 선택
  • #id : 해당 id를 지닌 요소 하나만 선택
  • .className: 해당 class값을 지닌 모든 요소 선택
  • * : 모든 요소 선택
  • > : 해당 요소의 자식요소 선택(정확하게 자식 요소만)
  • 공백: 해당 요소의 자손 요소 선택(내부에 잇는 모든 요소가 자손)
  • , : 여러개의 선택자를 한 번에 사용할 때 사용

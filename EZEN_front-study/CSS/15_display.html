# display 속성

  • 해당 요소가 어떻게 그려야 하는지 결정하는 속성
  • 요소가 그려지는 방식에는 기본적으로 block과 inline이 있다

# block 요소

  • 너비를 몇으로 설정하던지 항상 한 줄을 모두 차지한다
  • 기본으로 설정된 width는 100%이다
  • padding과 margin을 적용할 수 있다
  • 박스 취급을 당하는 요소이다
  • ex> div, p, h1~6, ul, li, ol, ... 들은 기본 display속성값이 block이다

# inline 요소

  • 영역을 지정하더라도 문단을 헤치지 않는다
  • 글 내용에서 특정 부분을 선택하기 위한 용도로 사용한다
  • padding과 margin이 올바르게 적용되지 않는다
  • 박스 취급이 아닌, 글자 취급을 당한다
  • ex> b, a, i, span, mark, ... 들은 기본 display 속성값이 inline이다

# display: inline-block

  • inline과 block의 장점을 합쳐놓은 display
  • 글 내용과 어우러지면서 padding과 margin도 적용할 수 있다

# display: none

  • 해당 요소를 보이지 않게 설정한다
  • 해당 요소가 차지하고 있던 영역도 사라진다

# visibility: hidden

  • 해당 요소의 visibility값을 hidden으로 설정하면 안보이게 된다
  • 해당 요소가 원래 차지하고 있던 영역은 그대로 차지하고 있게 된다

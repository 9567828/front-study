# clear속성과 함께 사용하기

  • clear: left - 해당 float 요소의 왼쪽에 다른 float요소가 없었으면 한다
  • clear: right - 해당 float 요소의 오른쪽에 다른 float요소가 없었으면 한다
  • clear: both - 해당 float 요소의 양쪽에 다른 float요소가 없었으면 한다

# 글 내용이 부족할 때 발생하는 문제 해결하기

  • float요소들이 요소의 크기에 잡히지 않아 문제가 발생하는 경우가 있다
  • 이 경우 clear를 설정한 요소를 맨 마지막에 두어 float을 마무리 짓고 요소의 전체범위를 인식하게 만들어줄 수 있다
  • 이 때 주로 ::after 가장 요소를 함께 활용한다

# 다른 페이지로 이동 할 수 있는 링크 걸기

- <>a태그 사용하기
- 자바스크립트 사용하기

# <>a태그 사용하기

- 자세한 내용을 알고 싶으시면 여기를 클릭!
- 여기(./naver.com)
- 여기(./www.naver.com)
- 여기(/www.naver.com)
- 여기(//www.naver.com)

# 웹에서 사용할 수 있는 경로의 종류(./, /, //)

- https://comic.naver.com/webtoon/list?titleId=697533가 있을 때
- 해당 위치에서 ./naver.com은 https://comic.naver.com/webtoon/naver.com으로 간다
  (가장 마지막 위치에 있는 /에서 해당 경로를 찾는다)
- 해당 위치에서 /naver.com은 https://comic.naver.com/naver.com으로 간다
  (가장 처음 위치에 있는 / 에서 해당 경로를 찾는다)
- 해당 위치에서 //naver.com http://naver.com으로 간다
  (프로토폴 다음에 있는 //에서 해당 경로를 찾는다)

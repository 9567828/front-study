# Document Object Model (DOM)

  • 웹 브라우저의 HTML코드를 해석하여 생성해둔 객체
  • DOM을 통해 HTML의 모든 요소에 접근할 수 있다

# 원하는 요소를 쉽게 선택하기 위한 document의 메서드들

  • document.getElementByID(id) : id로 원하는 요소를 선택한다
  • document.getElementTagName(tagName) : 해당 태그명의 모든 요소를 선택한다
  • document.getElementClassName(className) : 해당 클래스 속성을 지닌 모든 요소를 선택한다
  • document.querySelector(cssSelector) : CSS선택자를 활용해 원하는 요소 하나를 선택한다
  • document.querySelectorAll(cssSelector) : CSS선택자를 활용해 원하는 요소 여러개를 선택한다

# 선택한 HTML요소를 변경하기

  • element.innerHTML : 해당 요소의 내용을 변경한다 (tag 적용, 보안 위험함)
  • element.innerText : 해당 요소의 내용을 변경한다 (tag 미적용, 보안 좋음)
  • element.attribute : 해당요소의 속성을 직접 변경할 수 있다
  • element.setAttribute(): 해당 요소의 속성을 메서드로 변경할 수 있다
  • element.classList : 해당 요소의 클래스들을 리스트 형태로 관리하는 객체
  • element.style.CSSpropertyName : 해당 요소의 스타일 속성을 직접 변경할 수 있다. -가 포함된 property는 camelCase로 변경해야 한다 (e. g. border-width => borderWidth)

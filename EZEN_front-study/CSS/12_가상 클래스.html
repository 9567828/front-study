# Pseudo Class

  • 선택자에 추가로 사용하여 해당 요소의 특정 상황만을 선택하는 것
  • 선택자뒤에 :을 붙여 원하는 특정 상황을 고를 수 있다

# 종류

  • :first-child, :nth-child(n/even/odd), :last-child
  • :first-of-type, :nth-of-type(n/even/odd), :last-of-type
  • :hover, :link, :active, :visited
  • 이 외에 더 많은 가상 클래스들은 Pseudo Class로 구글링..

# 링크에 적용되는 가상 클래스 사용해보기

  • :link - 방문한 적이 없는 링크
  • :visited - 방문한 적이 있는 링크
  • :active - 링크를 클릭하고 있는 상태
  • :hover - 해당 요소에 마우스가 올라가 있을 때
